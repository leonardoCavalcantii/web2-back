# Game Rev-you

## Descrição 
Projeto Backend para a disciplina de Web 2 do cuso de Ciência da Computação@UFCG.
Sistema para review de jogos eletrônicos feitos por qualquer pessoa, sem a necessidade
de ser um crítico especialista em reviews de jogos.

    - Leonardo Andrade Cavalcanti Filho.
- Professor: José Glauber

## Setup

- Clonar o repositório:
```bash
git clone https://github.com/leonardoCavalcantii/web2-back.git
```

- Instalar dependências:
```bash
cd web2-back
npm install
```

- Configure seu Banco de Dados
- Defina uma string de conexão e defina de alguma forma como variável de 
ambiente do seu sistema operacional.

## Executar a aplicação web 

- npm run dev
